# بخش 9: سؤالات تحقیق

## سوال اصلی *(Bold — بدون شماره‌گذاری دو رقمی)*

چگونگی دسترسی به **هدف اصلی** را مورد سوال قرار می‌دهد.

## سوالات فرعی *(Bold — بدون شماره‌گذاری دو رقمی)*

- چگونگی دسترسی به هر **هدف فرعی** را می‌پرسد
- ترجیحاً ۲ یا ۳ سوال (هم‌تعداد با اهداف فرعی)

## نکته مهم

نگاشت سه‌طرفه باید کامل باشد:

```
هدف فرعی ۱  ←→  سوال فرعی ۱  ←→  فرضیه ۱
هدف فرعی ۲  ←→  سوال فرعی ۲  ←→  فرضیه ۲
```
