# بخش 7: اهداف تحقیق

## هدف اصلی *(Bold — بدون شماره‌گذاری دو رقمی)*

- معمولاً برابر عنوان پروپوزال است
- باید تفاوت تحقیق شما با کارهای قبلی مشخص باشد
- به صورت عبارت مصدری بنویس

## اهداف فرعی *(Bold — بدون شماره‌گذاری دو رقمی)*

- ۲ یا ۳ هدف قابل دسترس توسط دانشجو
- در راستای هدف اصلی (با رسیدن به آن‌ها، هدف اصلی تأمین شود)
- هر هدف: یک عبارت مصدری («طراحی ... کردن»، «ارزیابی ... کردن»، «مقایسه ... کردن»)

## ممنوع

- «کاهش هزینه سازمان‌ها»
- «گسترش استفاده از فناوری جدید»
- اهدافی که دانشجو نمی‌تواند مستقیماً به آن‌ها برسد

## مثال صحیح

**هدف اصلی:** طراحی و پیاده‌سازی مدل پیش‌بینی ... با استفاده از ... برای بهبود ...

**اهداف فرعی:**
- تعیین تأثیر ... بر دقت پیش‌بینی
- مقایسه عملکرد مدل پیشنهادی با روش‌های ... و ...
